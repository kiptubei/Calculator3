Calculator API

The Calculator3 Api allows you to perform some simple arithmetic, namely:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Factorial

Syntax

Addition

http://localhost:8080/Calculator3/start/calculate/add/{value1}/{value2}

  • http://localhost:8080/Calculator3/start/calculate/add/1/2

    • Subtraction

    http://localhost:8080/Calculator3/sub/{value1}/{value2}

  • http://localhost:8080/Calculator3/start/calculate/sub/5/3

    • Multiplication

    http://localhost:8080/Calculator3/start/calculate/mul/{value1}/{value2}

  • http://localhost:8080/Calculator3/start/calculate/mul/8/2

    • Divide

    http://localhost:8080/Calculator3/start/calculate/div/{value1}/{value2}

  • http://localhost:8080/Calculator3/start/calculate/div/9/78

    • Factorial

    Factorial takes one parameter only

    http://localhost:8080/Calculator3/start/calculate/fac/{value1}

  • http://localhost:8080/Calculator3/start/calculate/fac/5

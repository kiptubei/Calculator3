Calculator API

The calculator Api allows you to perform some simple arithmetic, namely:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Factorial

Syntax

Addition

http://localhost:8080/Calculator3/add/{value1}/{value2}

  • http://localhost:8080/Calculator3/add/1/2

    • Subtraction

    http://localhost:8080/Calculator3/subtract/{value1}/{value2}

  • http://localhost:8080/Calculator3/subtract/5/3

    • Multiplication

    http://localhost:8080/Calculator3/multiply/{value1}/{value2}

  • http://localhost:8080/Calculator3/multiply/8/2

    • Divide

    http://localhost:8080/Calculator3/divide/{value1}/{value2}

  • http://localhost:8080/Calculator3/divide/9/78

    • Factorial

    Factorial takes one parameter only

    http://localhost:8080/Calculator3/factorial/{value1}

  • http://localhost:8080/Calculator3/fact/5
